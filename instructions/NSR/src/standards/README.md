# NSR Standards

## Overview
This folder contains code and project standards for NSR-generated projects.

## Purpose
Standards provide:
- Code style guidelines (TypeScript, React, Tailwind CSS)
- Project structure standards
- File organization rules
- Documentation requirements
- Version control standards

## Standards

### `code-standards.md`
Code standards for NSR-generated projects. Includes:
- TypeScript standards
- React component patterns
- Styling guidelines (Tailwind CSS)
- File organization
- Documentation standards
- Error handling
- Performance optimization

### `project-standards.md`
Project structure and organization standards. Includes:
- Folder structure standards
- File organization rules
- Version control standards
- Task management standards
- Documentation standards
- Developer profiles

## Usage
Standards are used to:
- Enforce consistency across projects
- Guide code development
- Ensure proper project organization
- Maintain documentation quality
