# NSR Templates

## Overview
This folder contains templates for generating project structures and files using the NSR framework.

## Purpose
Templates provide reusable structures that can be used to generate:
- Project folder structures
- File organization patterns
- Documentation structures
- Configuration files

## Templates

### `project-structure-template.md`
Template for generating BitNexus-like project structures. Includes:
- Complete folder hierarchy
- Mandatory files and folders
- File organization rules
- Generation instructions

## Usage
Templates are used by:
- NSR project initialization workflow
- Project structure generators
- Documentation generators
