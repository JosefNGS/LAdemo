# NSR Rules

## Overview
This folder contains rules extracted from BitNexus and frameworks, organized for use in NSR-generated projects.

## Purpose
Rules provide:
- Cursor rules templates for generating `.cursorrules` files
- Extracted rules from BitNexus project
- Framework rules from Agent OS and BMAD-METHOD
- Project-specific rule patterns

## Rules

### `cursor-rules-template.md`
Template for generating `.cursorrules` files. Includes:
- Pre-flight checklist structure
- CHANGELOG requirements
- Critical rules sections
- Project structure rules
- Coding standards
- Service rules

## Usage
Rules are used to:
- Generate `.cursorrules` files for new projects
- Extract rules from existing projects
- Organize rules by category
- Apply standards to projects
