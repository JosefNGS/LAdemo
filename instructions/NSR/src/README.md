# NSR Framework Source

## Overview
This folder contains the source code and resources for the NSR (NorthStar Rules) framework. NSR is a project generation framework that enables the creation of BitNexus-like project structures.

## Purpose
The `src/` folder contains:
- **Templates** - Project structure templates for generating new projects
- **Rules** - Extracted rules from BitNexus and frameworks
- **Workflows** - Step-by-step workflows for project initialization
- **Standards** - Code and project standards for NSR-generated projects

## Structure
- `templates/` - Project structure templates
- `rules/` - Rules extraction and cursor rules templates
- `workflows/` - Project initialization workflows
- `standards/` - Code and project standards

## Usage
These files are used by NSR framework to:
- Generate project structures
- Create `.cursorrules` files
- Initialize new projects
- Enforce standards
