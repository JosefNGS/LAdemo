# NSR Workflows

## Overview
This folder contains step-by-step workflows for using the NSR framework to initialize and manage projects.

## Purpose
Workflows provide:
- Project initialization procedures
- Step-by-step guides for common tasks
- Automation instructions
- Best practices for using NSR

## Workflows

### `project-init-workflow.md`
Complete workflow for initializing a new project using NSR framework. Includes:
- Phase 1: Project Setup
- Phase 2: Rules & Standards
- Phase 3: Documentation
- Phase 4: Framework Integration
- Phase 5: Validation & Verification

## Usage
Workflows are used to:
- Guide project initialization
- Automate project setup
- Ensure consistency across projects
- Document best practices
