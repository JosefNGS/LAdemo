## Key Concepts

### BMAD Core
BMAD-METHOD is built on **BMAD Core** (Collaboration Optimized Reflection Engine):
- Universal framework for human-AI collaboration
- BMAD Method leverages Core for agile workflows
- BMAD Builder enables custom module creation

### Modules
- **BMB** (BMAD Method Builder)
- **BMGD** (BMAD Game Development)
- **BMM** (BMAD Method Main)
- **CIS** (Creative Intelligence System)

### Workflows
- Scale-adaptive (automatically adjusts planning depth)
- Complete lifecycle: Analysis → Planning → Architecture → Implementation
- Specialized agent expertise (PM, Architect, Developer, UX Designer, etc.)
