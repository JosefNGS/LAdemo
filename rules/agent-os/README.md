# Agent OS Framework Rules

## Overview

This directory contains rules extracted from the Agent OS framework located in `instructions/.agent-os/`.

Agent OS is a framework for AI-driven development that provides:
- Structured workflows for product planning and development
- Code style standards and best practices
- Security principles and guidelines
- Deployment requirements and standards

## Rule Files

- **`code-style.md`** - Code formatting, naming conventions, and style guidelines
- **`best-practices.md`** - Development principles (DRY, simplicity, readability)
- **`security.md`** - Core security principles and requirements
- **`deployment.md`** - Deployment requirements and infrastructure standards

## Source

Original documentation: `instructions/.agent-os/`

## When to Use

- When writing code that follows Agent OS standards
- When implementing security-sensitive features
- When planning deployments
- When establishing code style for new features
