# Netlify Functions

## Overview
This folder contains Netlify serverless functions for the BitNexus platform.

## Functions

- **submit-email.js** - Email submission handler
- **submit-email-airtable.js** - Email submission handler for Airtable
- **submit-email-supabase.js** - Email submission handler for Supabase/PostgreSQL

## Purpose

This folder houses all Netlify serverless functions that handle backend API requests.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Functions should follow Netlify serverless function best practices
- Functions should be properly documented
