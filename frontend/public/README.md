# Frontend Public Assets

## Overview
This folder contains static public assets for the BitNexus frontend application.

## Contents

- **`_redirects`** - Netlify redirect rules for SPA routing

## Purpose

This folder houses all static public assets that are served directly by the web server without processing.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Static assets should be optimized for production
- Redirect rules should be kept up-to-date with routing changes
