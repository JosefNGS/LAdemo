# Frontend Services

## Overview
This folder contains service layer code for API calls and business logic in the BitNexus frontend application.

## Services

- Service files for API integration, data fetching, and business logic

## Purpose

This folder houses all service layer code that handles API calls, data fetching, and business logic separate from UI components.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Services should follow TypeScript best practices
- API integration should be properly typed
