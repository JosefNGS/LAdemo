# Frontend Utils

## Overview
This folder contains utility functions and helpers for the BitNexus frontend application.

## Utility Files

- **marketingAssets.ts** - Marketing assets utility functions
- **profileNavigation.ts** - Profile navigation utility functions

## Purpose

This folder houses utility functions and helpers that are used across multiple components in the frontend application.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Utility functions should be properly typed with TypeScript
- Functions should be well-documented
