# Frontend Contexts

## Overview
This folder contains React context providers for the BitNexus frontend application.

## Context Files

- **CartContext.tsx** - Shopping cart context provider
- **NotificationContext.tsx** - Notification context provider

## Purpose

This folder houses React context providers that manage global state across the frontend application.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Contexts should follow React best practices
- Context providers should be properly typed with TypeScript
