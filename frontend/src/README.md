# Frontend Source Code

## Overview
This folder contains all React/TypeScript source code for the BitNexus frontend application.

## Structure

- **`components/`** - Reusable React components
- **`pages/`** - Page-level React components
- **`services/`** - Service layer for API calls and business logic
- **`data/`** - Mock data and data structures
- **`utils/`** - Utility functions and helpers
- **`contexts/`** - React context providers
- **`types.ts`** - TypeScript type definitions
- **`constants.tsx`** - Application constants
- **`main.tsx`** - Application entry point
- **`App.tsx`** - Main application component

## Purpose

This folder houses all frontend source code, including React components, TypeScript types, services, utilities, and application logic.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Code should follow TypeScript best practices
- Components should be organized by functionality
- All UI/UX changes are managed by Josef Lindbom
