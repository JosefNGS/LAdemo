# Frontend Pages

## Overview
This folder contains page-level React components for the BitNexus frontend application.

## Pages

This folder contains all page-level components that represent different routes in the application, including:
- Dashboard, Marketplace, Alliance, Bot Lab, Earn, Content Generator
- Admin View, Affiliate Manager, News, Feed, Chat, Friends
- User Profile, Team Profile, Token Shop, Academy, and more

## Purpose

This folder houses all page-level React components that represent different views/routes in the application.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Pages should follow React best practices
- All UI/UX changes are managed by Josef Lindbom
- Pages should be properly typed with TypeScript
