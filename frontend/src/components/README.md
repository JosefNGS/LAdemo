# Frontend Components

## Overview
This folder contains reusable React components for the BitNexus frontend application.

## Components

- **AdminLoginModal.tsx** - Admin login modal component
- **GettingStartedModal.tsx** - Getting started modal component
- **Layout.tsx** - Main layout component
- **NotificationBell.tsx** - Notification bell component
- **PlatformAdminUsers.tsx** - Platform admin users component
- **ProductDetailDrawer.tsx** - Product detail drawer component
- **ProductUploadForm.tsx** - Product upload form component
- **TeamTaskChecklist.tsx** - Team task checklist component

## Purpose

This folder houses all reusable React components that can be used across multiple pages in the application.

## Maintenance

- All changes must be logged in `CHANGELOG.md`
- Components should follow React best practices
- All UI/UX changes are managed by Josef Lindbom
- Components should be properly typed with TypeScript
