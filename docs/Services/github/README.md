# GitHub Service Documentation

## Overview
This folder contains documentation for GitHub version control, repository management, and developer collaboration workflows.

## Purpose
Documentation for:
- Version control workflows
- Branch protection rules
- Pull request processes
- Developer registry
- Push change documentation
- Repository management

## Key Files

### Main Documentation
- **`SERVICE_RULES.md`** - GitHub service-specific rules
- **`VERSION_CONTROL.md`** - Git workflow guidelines
- **`DEVELOPERS.md`** - Active developer registry
- **`GITHUB_PUSH_INSTRUCTIONS.md`** - Push workflow instructions
- **`PUSH_CHANGE_DOCUMENTATION_TEMPLATE.md`** - Template for push documentation

### Change Documentation
- **`push-docs/`** - Push change documentation (one file per push)
