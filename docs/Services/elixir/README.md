## Overview

Elixir services handle business logic, concurrency, and integration with the Erlang/Elixir blockchain ledger using BEAM VM processes. Phoenix web framework calls Elixir services for business logic.

## Key Features

- **BEAM VM**: Process isolation and fault tolerance
- **GenServer**: Stateful processes
- **Supervisor**: Process supervision trees
- **Ecto**: Database access
- **Concurrency**: Millions of lightweight processes
