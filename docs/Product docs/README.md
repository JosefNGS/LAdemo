# Product Documentation

This directory contains all product-related documentation organized by topic.

## Structure

- **`Business & Strategy/`** - Business planning and strategy documents
- **`Investor Ready/`** - Investor-ready documentation (15 folders)
- **`Legal & Compliance/`** - Legal protection documentation
- **`Pitch Deck & Presentations/`** - Pitch decks and presentation materials
- **`Technical Documentation/`** - Technical specs and documentation
- **`Tokenomics/`** - Tokenomics and credit system

## BitNexusDocs Repository Sync

**CRITICAL**: When ANY file in `docs/Product docs/` is updated, it MUST be synced to BitNexusDocs repository.

- **Repository**: [https://github.com/JosefNGS/BitNexusDocs](https://github.com/JosefNGS/BitNexusDocs)
- **Sync Process**: See `docs/Services/github/BITNEXUS_DOCS_SYNC.md`
- **When to Sync**: After every update to Product docs, before sharing with stakeholders
