# 03_Financial_Model

## Purpose
This folder contains all financial projections, models, and historical financial data.

## Required Documents

### Master Financial Model (Excel)
- **Status**: ⚠️ To be created
- **Purpose**: Comprehensive financial model with multiple scenarios
- **Required Tabs**:
  - **Assumptions**: Key variables and assumptions (growth rates, conversion rates, pricing, etc.)
  - **P&L**: Profit and Loss statements (monthly/quarterly/annual)
  - **Balance Sheet**: Assets, liabilities, and equity projections
  - **Cash Flow**: Operating, investing, and financing cash flows
- **Usage**: Primary tool for financial analysis and scenario planning

### Historical Financials
- **Status**: ⚠️ To be created (if applicable)
- **Purpose**: P&L for previous years (if company has operating history)
- **Content**: Actual financial performance from past periods
- **Usage**: Demonstrate historical performance and trends

### Cap Table (Excel)
- **Status**: ⚠️ To be created
- **Purpose**: Current ownership structure
- **Content**: 
  - Current shareholders and their ownership percentages
  - Option pool details
  - Convertible notes and their conversion terms
  - Reserved shares for future issuance
- **Usage**: Show current equity structure and dilution impact
- **Note**: May be placed in `08_Company_Docs` if preferred

## Notes
- Financial models should be well-documented with clear assumptions
- Include sensitivity analysis for key variables
- Keep models updated with latest actuals and revised projections
- Use consistent formatting and clear labels
