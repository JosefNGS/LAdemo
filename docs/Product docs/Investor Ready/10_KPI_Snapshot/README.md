# 10_KPI_Snapshot

## Purpose
This folder contains key performance indicators, metrics dashboards, and cohort analysis.

## Required Documents

### Key Metrics Dashboard
- **Status**: ⚠️ To be created
- **Purpose**: Retention curves, Churn rates, DAU/MAU, ARR/MRR growth
- **Content**: 
  - User metrics (DAU, MAU, retention)
  - Revenue metrics (ARR, MRR, growth rates)
  - Engagement metrics
  - Conversion metrics
  - Churn analysis
- **Format**: Dashboard screenshots or Excel/PDF reports
- **Usage**: Show business health and growth trajectory

### Cohort Analysis
- **Status**: ⚠️ To be created
- **Purpose**: How user groups behave over time
- **Content**: 
  - User cohort retention curves
  - Revenue cohort analysis
  - Behavior patterns by cohort
  - Cohort-based growth metrics
- **Usage**: Demonstrate user value and retention patterns

## Key Metrics to Track
- **User Metrics**: DAU, MAU, retention rates, churn
- **Revenue Metrics**: ARR, MRR, growth rates, LTV
- **Engagement**: Active users, session duration, feature usage
- **Conversion**: Sign-up to paid, trial to paid, referral rates
- **Product Metrics**: Feature adoption, NPS, customer satisfaction

## Notes
- Metrics should be updated regularly (weekly/monthly)
- Include both leading and lagging indicators
- Show trends over time, not just point-in-time snapshots
- Be transparent about both positive and negative trends
