# 11_Costs

## Purpose
This folder contains cost structure, vendor contracts, and salary information.

## Required Documents

### Vendor Contracts
- **Status**: ⚠️ To be created
- **Purpose**: Major software or service agreements (AWS, Lease, etc.)
- **Content**: 
  - Key vendor contracts and agreements
  - Service level agreements
  - Pricing and terms
  - Contract expiration dates
- **Usage**: Show cost structure and vendor relationships

### Salary Breakdown
- **Status**: ⚠️ To be created
- **Purpose**: Anonymized census of employee salaries
- **Content**: 
  - Salary ranges by role
  - Total compensation costs
  - Benefits and equity costs
  - Hiring budget projections
- **Format**: Anonymized spreadsheet or summary document
- **Usage**: Understand cost structure and hiring plans

## Cost Categories
- **Infrastructure**: Hosting, cloud services, development tools
- **Software/SaaS**: Subscriptions, licenses, tools
- **Personnel**: Salaries, benefits, equity
- **Marketing**: Advertising, content, events
- **Operations**: Office, legal, accounting, insurance
- **R&D**: Development tools, testing, research

## Notes
- Vendor contracts should be summarized (not full contracts unless requested)
- Salary information should be anonymized and aggregated
- Include both current costs and projected costs
- Show cost efficiency and optimization efforts
