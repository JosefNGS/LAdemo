# 07_Team_Bios

## Purpose
This folder contains information about the team, leadership, and advisors.

## Required Documents

### Leadership Resumes/CVs
- **Status**: ⚠️ To be created
- **Purpose**: PDF versions for all founders and key execs
- **Content**: 
  - Professional backgrounds
  - Relevant experience
  - Education and credentials
  - Key achievements
- **Format**: Individual PDF files for each team member
- **Usage**: Demonstrate team capability and experience

### Org Chart
- **Status**: ⚠️ To be created
- **Purpose**: Current structure and hiring roadmap for the next 12–18 months
- **Content**: 
  - Current organizational structure
  - Roles and responsibilities
  - Reporting relationships
  - Hiring plan and timeline
  - Open positions and priorities
- **Usage**: Show organizational maturity and growth plans

### Advisor/Board List
- **Status**: ⚠️ To be created
- **Purpose**: Bios of key advisors or board members
- **Content**: 
  - Advisor names and roles
  - Professional backgrounds
  - Areas of expertise
  - How they contribute to the company
- **Usage**: Demonstrate access to expertise and networks
