# Core Documentation

This directory contains core system documentation including project structure, documentation organization, and technology stack.

## Files

- **`STRUCTURE.md`** - Main project structure reference (AUTHORITATIVE SOURCE)
- **`DOCS_STRUCTURE.md`** - Documentation organization guide
- **`TECH_STACK.md`** - Technology stack documentation

## Purpose

- **STRUCTURE.md**: Defines the complete project structure - all files must follow this structure
- **DOCS_STRUCTURE.md**: Defines how documentation is organized
- **TECH_STACK.md**: Documents the technology stack and dependencies

## Usage

- **Before creating files**: Check `STRUCTURE.md` to determine correct location
- **Before organizing docs**: Check `DOCS_STRUCTURE.md` for organization rules
- **Before adding dependencies**: Check `TECH_STACK.md` for tech stack standards
